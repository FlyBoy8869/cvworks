# CVWorks

An interactive calendar indicating whether CV is working or off.<br>
<br>
Just click a date and see what you get.
<br><br>
### Keyboard Shortcuts:

* CTRL-A - About<br>
* CTRL-R - Reset screen position and dimensions<br>
* CTRL-T - Jump to today's date<br>
